Limited Integration with Basic Link

Limited integration is a convenient option for services that want to use SeamlessAccess discovery service but have limited abilities to change existing login button or link.

If you cannot customise an existing login link, that is fine. All you need to do is to point it to SeamlessAccess discovery service.

Check out this demo to experience user login to a website that implements SeamlessAccess discovery service behind an existing login link.
